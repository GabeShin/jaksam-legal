# 작샘 (Jaksam) 개인정보처리방침

**시행일**: 2026년 2월 12일
**최종 수정**: 2026년 3월 5일

---

## 1. 개요

작샘(이하 "앱")은 음악 학생, 교사, 학부모를 위한 레슨 녹음 및 연습 관리 서비스입니다. 본 개인정보처리방침은 앱이 수집, 이용, 보관, 공유하는 개인정보에 대해 설명합니다.

앱을 사용함으로써 본 방침에 동의하는 것으로 간주됩니다.

---

## 2. AI 데이터 처리에 대한 동의

앱은 핵심 기능을 제공하기 위해 인공지능(AI) 기술을 사용합니다. **앱을 처음 사용할 때 AI 데이터 처리에 대한 명시적 동의를 요청하며, 동의하지 않으면 서비스를 이용할 수 없습니다.**

AI 처리에 포함되는 데이터:
- 레슨 및 연습 녹음의 오디오/동영상 (텍스트 변환용)
- 전사된 텍스트 (요약, 연습 계획, 피드백 질문 생성용)
- 학생 프로필 정보 (개인화된 AI 코칭용)
- 연습 기록 및 피드백 응답 (맞춤형 조언 생성용)
- AI 코치와의 대화 내용 (대화 맥락 유지 및 응답 생성용)

이 데이터는 미국에 소재한 **OpenAI**의 API를 통해 처리됩니다. OpenAI API를 통해 전송된 데이터는 모델 학습에 사용되지 않습니다 (OpenAI API 이용약관 기준).

동의는 앱 내 설정에서 언제든지 철회할 수 있으며, 철회 시 AI 기능이 비활성화됩니다.

## 3. 수집하는 개인정보

### 3.1 회원가입 시 수집하는 정보

| 항목 | 필수 여부 | 목적 |
|------|----------|------|
| 이메일 주소 | 필수 | 계정 생성, 로그인, 비밀번호 재설정 |
| 비밀번호 | 필수 | 계정 인증 (암호화 저장) |
| 이름 (표시 이름) | 필수 | 앱 내 프로필 표시 |
| 역할 (학생/교사/학부모) | 필수 | 역할별 기능 제공 |

### 3.2 소셜 로그인 시 수집하는 정보

카카오 또는 Apple 로그인을 사용하는 경우, 해당 서비스로부터 다음 정보를 제공받습니다:
- 계정 식별자
- 이메일 주소 (계정에 등록된 경우)

### 3.3 학생 프로필 정보

학생은 온보딩 과정 또는 설정에서 다음 정보를 선택적으로 제공할 수 있습니다:

| 항목 | 목적 |
|------|------|
| 생년월일 | 연령에 맞는 AI 코칭 제공 |
| 악기 종류 | 악기별 맞춤 연습 계획 생성 |
| 연주 경력 (년) | 수준에 맞는 피드백 제공 |
| 일일/주간 연습 목표 | 연습 진행률 추적 및 리마인더 |
| 선호 연습 시간대 | 알림 시간 최적화 |

이 정보는 AI가 학생 수준에 맞는 개인화된 피드백과 연습 계획을 생성하는 데 사용됩니다.

### 3.4 서비스 이용 중 수집하는 정보

| 항목 | 설명 |
|------|------|
| 레슨 녹음 (오디오/동영상) | 학생이 직접 녹음한 음악 레슨 |
| 연습 녹음 (오디오/동영상) | 학생이 직접 녹음한 연습 세션 |
| 프로필 사진 | 선택적으로 업로드하는 프로필 이미지 |
| 연습 기록 | 연습 시간, 빈도, 피드백 응답 |
| 학습 목표 | 학생이 설정한 연습 목표 및 기한 |
| 피드백 응답 | 연습 후 질문에 대한 학생의 응답 |
| AI 코치 대화 | AI 코치(작샘)와의 채팅 메시지 |

### 3.5 자동으로 수집하는 정보

| 항목 | 목적 |
|------|------|
| 기기 식별 토큰 | 푸시 알림 전송 |

오류 보고에는 기기 유형, OS 버전, 앱 버전, 오류 발생 시점의 화면 정보가 포함될 수 있습니다. 녹음 내용이나 개인 메시지는 오류 보고에 포함되지 않습니다.

## 4. 개인정보의 이용 목적

수집한 개인정보는 다음 목적으로 이용됩니다:

1. **서비스 제공**: 계정 관리, 레슨/연습 녹음 및 재생, 연습 기록 관리
2. **AI 텍스트 변환**: 녹음된 오디오/동영상을 텍스트로 변환 (OpenAI Whisper API)
3. **AI 요약 및 계획 생성**: 레슨 요약, 연습 계획, 피드백 질문 생성 (OpenAI GPT API)
4. **AI 코칭**: 개인화된 연습 조언, 일일 연습 계획, 주간 체크인 메시지 제공 (OpenAI GPT API)
5. **알림**: 레슨 처리 완료, 연습 리마인더, AI 코칭 메시지 등 푸시 알림 전송
6. **오류 모니터링**: 앱 충돌 및 오류 감지, 서비스 안정성 개선
7. **교사/학부모 연동**: 초대 코드를 통한 교사-학생, 학부모-학생 연결 및 데이터 공유

## 5. 개인정보의 제3자 제공 및 위탁

앱은 서비스 제공을 위해 다음 제3자 서비스를 이용합니다:

| 서비스 제공자 | 이용 목적 | 제공 데이터 |
|--------------|----------|------------|
| **Supabase** (미국) | 인증, 데이터베이스, 파일 저장 | 계정 정보, 녹음 파일, 모든 서비스 데이터 |
| **OpenAI** (미국) | 음성 전사 (Whisper API), 요약/연습계획/코칭 생성 (GPT API) | 녹음 오디오, 전사된 텍스트, 학생 프로필, 피드백 응답, AI 코치 대화 내용 |
| **Sentry** (미국) | 오류 및 성능 모니터링 | 오류 정보, 기기 정보, 사용자 ID 및 역할 |
| **카카오** (대한민국) | 소셜 로그인 (선택 사용 시) | 인증 토큰 |
| **Apple** (미국) | Apple 로그인 (선택 사용 시) | 인증 토큰 |
| **Expo** (미국) | 푸시 알림 | 기기 푸시 토큰 |

### OpenAI 데이터 처리에 관한 상세 안내

다음 데이터가 OpenAI API로 전송됩니다:

- **음성 전사**: 레슨 및 연습 녹음의 오디오가 텍스트 변환을 위해 OpenAI Whisper API로 전송됩니다.
- **요약/계획 생성**: 전사된 텍스트가 요약 및 연습 계획 생성을 위해 OpenAI GPT API로 전송됩니다.
- **AI 코칭**: 학생 프로필(연령, 악기, 경력), 최근 연습 기록, 피드백 응답, 이전 대화 내용이 개인화된 코칭 메시지 생성을 위해 OpenAI GPT API로 전송됩니다.
- **주간 체크인**: 주간 연습 데이터와 레슨 요약이 체크인 질문 생성을 위해 OpenAI GPT API로 전송됩니다.

OpenAI API는 전송된 데이터를 모델 학습에 사용하지 않습니다 (OpenAI API 이용약관 기준).

개인정보를 제3자에게 판매하지 않습니다.

## 6. 개인정보의 보관 및 보호

### 6.1 보관 기간

- 계정 정보: 회원 탈퇴 시까지
- 녹음 파일 및 관련 데이터 (전사, 요약, 연습 계획): 사용자가 삭제하거나 회원 탈퇴 시까지
- AI 코치 대화 기록: 사용자가 삭제하거나 회원 탈퇴 시까지
- 오류 로그: 수집 후 90일

### 6.2 보안 조치

- 모든 데이터는 전송 시 TLS/SSL 암호화 적용
- 비밀번호는 단방향 암호화 (해싱) 저장
- 데이터베이스 접근은 행 수준 보안(RLS) 정책으로 제한
- 녹음 파일 접근은 서명된 URL (만료 시간 포함)로 제한
- 서버 측 서비스 키는 클라이언트에 노출되지 않음

## 7. 이용자의 권리

이용자는 다음의 권리를 행사할 수 있습니다:

1. **정보 열람**: 앱 내 설정에서 본인의 개인정보를 확인할 수 있습니다.
2. **정보 수정**: 이름, 프로필 사진 등을 수정할 수 있습니다.
3. **녹음 삭제**: 개별 레슨 및 연습 녹음을 삭제할 수 있습니다.
4. **계정 삭제**: 앱 내 설정에서 계정 삭제를 요청할 수 있습니다. 요청 후 30일의 유예 기간이 적용되며, 유예 기간 내에 로그인하면 삭제가 취소됩니다. 유예 기간 종료 후 모든 개인정보와 녹음 데이터가 영구 삭제됩니다.
5. **연동 해제**: 교사 또는 학부모와의 연결을 언제든지 해제할 수 있습니다.
6. **AI 동의 철회**: AI 데이터 처리에 대한 동의를 철회할 수 있습니다.

계정 삭제를 원하시면 앱 내 설정 > 프로필 편집 > 계정 삭제 또는 아래 연락처로 요청해 주세요.

## 8. 교사 및 학부모의 데이터 접근

- **교사**: 초대 코드로 연결된 학생의 레슨 녹음, 전사, 요약, 연습 계획, 연습 기록을 열람할 수 있습니다. 교사는 연습 계획 항목을 수정할 수 있습니다.
- **학부모**: 초대 코드로 연결된 자녀의 레슨 및 연습 데이터를 열람할 수 있습니다 (읽기 전용).
- 연결은 학생이 생성한 초대 코드를 통해서만 가능하며, 학생이 언제든지 해제할 수 있습니다.
